# Galaxy Machine Learning Project

**Data Source:**
This project uses the Galaxy Zoo dataset.
1. Download `images_training_rev1.zip` and `training_solutions_rev1.csv` from [Kaggle](https://www.kaggle.com/c/galaxy-zoo-the-galaxy-challenge/data).

2. Extract them into the `data/` folder.

3. Run `src/1_prepare_data.py` to generate the clean dataset.
